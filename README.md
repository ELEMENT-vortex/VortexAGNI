# Vortex Kernel

**Kernel para Redmi Note 10 Pro (sweet/sweetin)**

Base: AGNI 1.4 | API: Vortex API v2.2 | ROM: AOSP / Android 16 (SDK 36)

## Descargas

Ve a [Releases](../../releases) para descargar el kernel ZIP.

## Modulo KSU

Instala tambien [VORTEX-exchange](https://github.com/ELEMENT-vortex/VORTEX-exchange) desde KernelSU-Next.

## Especificaciones

- Dispositivo: Redmi Note 10 Pro (sweet / sweetin)
- Version del kernel: \4.14.357-VortexAGNI-1.4\
- Vortex API v2.2 , 28 nodos sysfs (SAFE + P1)
- KernelSU integrado
- AOSP / Android 16 / SDK 36
